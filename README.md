# 📚 Library Management System (Backend)

A robust and secure backend library management system developed using Node.js and MySQL. This project dynamically handles core CRUD operations—such as adding, deleting, searching, and updating books—directly through the terminal interface.

## 🚀 Features
- **Full CRUD Support:** Dedicated modular files for adding (`add.js`), searching (`search.js`), updating (`update.js`), and deleting (`delete.js`) books.
- **Secure Architecture:** Sensitive database credentials and configuration keys are securely managed using `.env` (Environment Variables) to prevent security leaks.
- **Modular Codebase:** Each database operation is separated into functional modules, adhering to clean code and sustainability principles.

## 🛠️ Tech Stack
- **Runtime Environment:** Node.js
- **Database:** MySQL
- **Package Management:** npm (dependencies: dotenv, mysql2, etc.)
- **Version Control:** Git & GitHub
